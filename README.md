# tooltip
输入框提示框插件

# 使用方法：
例（html）:
```
<a id="qiuye-top">鼠标悬浮上面显示提示</a>
```
js:
```
$("#qiuye-top").tooltip({
	position: "top",
	content: '这里放文本',
	background: "#3c4751",
	event: "hover",
	color: "yellow"
});
```
无需另外引用css！
